# Underwater Fish Tracking System

## Project Overview

The Underwater Fish Tracking System is an AI-powered computer vision project designed to detect and track fish in underwater environments. This solution helps researchers and marine conservationists monitor fish behavior, movement patterns, and population trends efficiently.

## Features

- Real-time fish detection
- Fish tracking across video frames
- Computer vision-based analysis
- Automated monitoring system
- Marine ecosystem observation

## Technologies Used

- Python
- OpenCV
- NumPy
- YOLO
- Computer Vision

## Applications

- Marine Research
- Fish Population Monitoring
- Aquaculture Management
- Environmental Conservation

## Workflow

1. Input underwater video is provided.
2. YOLO model detects fish in each frame.
3. Fish are assigned unique tracking IDs.
4. Fish movement is tracked across frames.
5. Total fish count is calculated and displayed.

## Results

- Successfully detected fish in underwater videos.
- Tracked fish movement accurately.
- Counted fish in real-time.
- Improved monitoring efficiency.
